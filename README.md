# NAI Sync Standalone (NovelAI Local Panel)

NovelAIの画像生成画面に、生成履歴の自動保存やプロンプトの復元、タグのオートコンプリート機能などを追加するTampermonkeyスクリプトです。

本スクリプトはブラウザ内蔵のデータベース（IndexedDB）を利用するため、**外部のサーバー等を一切必要とせず、スマホなどのローカル環境単独で動作**します。

## 主な機能
- **自動履歴保存**: 画像を生成するたびに、ブラウザのIndexedDBに画像とプロンプト・シード値などの設定が自動で保存されます。
- **履歴ギャラリー**: セッションごとに整理された履歴フォルダ、および画像ごとの詳細なパラメータを確認できます。
- **プロンプト復元機能**: 履歴の画像やタイムスタンプをクリック・タップするだけで、当時のプロンプトやシード値をUI上に一瞬で復元します。
- **お気に入り機能**: 履歴画面の「☆」ボタンでお気に入り登録し、後からいつでも呼び出せます。
- **オフラインデータ管理**: 履歴全体を一つのZIPファイル(JSONデータ)としてダウンロードし、バックアップできます。バックアップはローカルデータを上書きすることなく、「プレビューモード」としていつでも閲覧・復元可能です。
- **タグオートコンプリート機能**: プロンプト入力時に Danbooru や e621 のタグ候補をサジェスト表示します。
- **画像一覧グリッド表示**: 画面上のボタンから、現在生成されている画像を一覧でグリッド表示し、素早く確認することができます。
- **プロンプト強調ショートカット**: プロンプト入力欄にて十字キー（↑・↓）を使用することで、プロンプトの強調値（括弧 `{ }` や `[ ]` の増減）を直感的に調整できます。

## インストール方法

1. お使いのブラウザに拡張機能 [Tampermonkey](https://www.tampermonkey.net/) をインストールします。
2. 以下のリンクをクリックして、スクリプトをインストールします。
   👉 **[スクリプトをインストール](https://raw.githubusercontent.com/vqMP7pwqBefqkoG/NAI-sync-standalone/main/NovelAI_Local.user.js)**
3. NovelAIの画像生成画面（`https://novelai.net/image`）を開くと、画面右端（または設定した位置）に「📜 履歴」というタブが表示されます。

## 使い方

1. **基本操作**:
   NovelAI上で通常通りに画像を生成してください。生成完了時、自動的に右側の「履歴」タブの中に画像がストックされていきます。
2. **履歴の閲覧**:
   「📜 履歴」タブをクリック（タップ）するとパネルが開き、生成履歴を確認できます。フォルダを開くと、そのセッションで生成した画像の一覧が見られます。
3. **設定の復元**:
   履歴内の「画像」や「時間表示」をクリック（タップ）すると、その画像を生成した時点のプロンプト・ネガティブプロンプト・シード値・サンプラーなどの全設定が画面に入力されます。
4. **連続生成 (バッチ)**:
   パネル下部の「連続生成」から回数を指定して開始すると、スリープを防止しながら指定回数分の自動生成を行います。

### データのバックアップについて
ブラウザのキャッシュクリア等でIndexedDBのデータが消失するのを防ぐため、定期的なバックアップを推奨します。

- パネル下部の「💾 データ管理」ボタンを押します。
- **📥 バックアップをダウンロード** を押すと、現在の全履歴とお気に入りが1つのZIP(JSON)としてダウンロードされます。
- **👁️ バックアップをプレビュー** から保存したファイルを選択すると、現在のデータを上書きすることなく、バックアップ当時の履歴・お気に入りを一時的にプレビューできます。プレビュー中に画像をタップしてプロンプトを復元することも可能です。

## 注意事項
- 初回起動時、またはタグ辞書が未キャッシュの場合、裏側でタグデータ（Danbooru/e621）を数MB分ダウンロードします。初回はプロンプトのオートコンプリートが効き始めるまで数秒〜数十秒かかる場合があります。
- ブラウザの「Cookieやサイトデータを削除する」機能を実行すると、IndexedDBの履歴データも一緒に消去されますのでご注意ください。

## 開発・ビルド情報
本スクリプトは `NovelAI_Sync.user.js` をベースに `build_local.js` を用いて、スタンドアローン環境向けに動的変換・ビルドされています。
